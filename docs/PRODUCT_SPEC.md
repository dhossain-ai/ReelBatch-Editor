# ReelBatch Editor — Product Spec

## Product Name

ReelBatch Editor

## One-Line Description

A batch video editing tool for reels with blur, logo overlay, zoom/crop, and automated export.

## Target Users

* Facebook Reels creators
* TikTok/Shorts creators
* AI video creators
* Social media page managers
* Small content teams that process many short videos

## Problem

Creators often need to apply the same edit to many short videos, such as hiding a fixed logo area, adding their own branding, or slightly zooming/cropping the video. Doing this manually one by one in CapCut or similar editors is slow.

## Solution

ReelBatch Editor lets users import many videos, choose the target area once, select an operation, and export all processed videos automatically.

## MVP Editing Modes

### 1. Blur Selected Area

The user draws a rectangle over the logo/watermark area. The app blurs that area for every imported video.

### 2. Cover With Logo/Image

The user selects an area and uploads a logo/image. The app scales the logo/image to the selected area and overlays it on every video.

### 3. Zoom/Crop

The user chooses a zoom percentage. The app scales the video slightly and crops it back to the original vertical format.

## Main Workflow

1. Open app.
2. Import multiple MP4 videos.
3. Select output folder.
4. Preview first video.
5. Draw rectangle over target area.
6. Choose edit mode.
7. Adjust settings.
8. Click Export All.
9. App processes all videos and saves outputs.

## Success Criteria

The MVP is successful if the user can process at least 10 vertical MP4 reels automatically without manually editing each video one by one.
